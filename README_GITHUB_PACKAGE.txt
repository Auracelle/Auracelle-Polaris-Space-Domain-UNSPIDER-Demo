QUICK START: 3 STEPS TO DEPLOY
================================================================================

Step 1: Create a GitHub repository
   - Go to github.com/new
   - Name: Auracelle-Polaris-UN-SPIDER
   - Public: Yes
   - Clone it locally

Step 2: Copy all 6 files (+ .nojekyll) to your cloned repository

Step 3: Push to GitHub
   git add .
   git commit -m "Initial commit: Auracelle Polaris UN-SPIDER Edition"
   git push

That's it! GitHub Pages automatically deploys within 1 minute.

Your live URL will be:
https://YOUR-USERNAME.github.io/Auracelle-Polaris-UN-SPIDER/
